# ☁️ Azure Blob Storage API (.NET 10)

Esta aplicação é uma **Azure Function** (Isolated Worker) desenvolvida com **C# 14** e **.NET 10**. Ela fornece uma interface via **Swagger** para gerenciar arquivos (upload, download e listagem) no Azure Blob Storage.

## 🛠️ Tecnologias
- **Runtime:** Azure Functions v4 (Isolated)
- **Framework:** .NET 10
- **Storage:** Azure Blob Storage SDK
- **Documentação:** Swagger / OpenAPI 3.0
